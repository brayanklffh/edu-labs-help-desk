Edu Labs Help Desk 
Este proyecto es un prototipo funcional de un sistema de mesa de ayuda (Help Desk) diseñado para gestionar tickets de soporte técnico de manera eficiente. El sistema integra una interfaz intuitiva con lógica de programación para la categorización y gestión de solicitudes.

Características Principales
Gestión de Tickets: Permite la creación y visualización de solicitudes de soporte en tiempo real.

Asistente Virtual: Módulo de chat integrado para brindar respuestas automáticas y apoyo al usuario.

Diseño Profesional: Interfaz sobria y elegante, alejada de estéticas de videojuegos, enfocada en un entorno corporativo.

Arquitectura Limpia: Estructura de archivos organizada en la raíz para facilitar el despliegue y mantenimiento.

Tecnologías Utilizadas
HTML5: Estructura semántica del sitio.

CSS3: Estilos avanzados utilizando Flexbox y CSS Grid.

JavaScript (ES6+): Lógica de negocio centrada en manipulación del DOM y métodos de arreglos como .map(), .filter() y .find().

Metodología BEM: Para un código CSS escalable y organizado.

Mobile First: Garantizando que la interfaz sea totalmente responsiva.

Estructura del Proyecto
index.html: Punto de entrada principal de la aplicación.

Style.css: Hoja de estilos con diseño responsivo y profesional.

desplegue.js: Lógica funcional para la administración de tickets y el chat.

README.md: Documentación del proyecto.
